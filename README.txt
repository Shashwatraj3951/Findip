Documentation available @ https://github.com/Shashwatraj3951/Findip

This tool is tested on :--

1. Kali Linux
2. Parrot
3. Ubuntu
4. Termux

This tool helps you to find real IP address of your target with just a single click.

Installtion:-

1. apt get update
2. apt get install git curl -y
3. git clone https://github.com/Shashwatraj3951/Findip.git
4. cd Findip
5. bash findip
